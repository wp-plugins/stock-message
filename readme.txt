WooCommerce plugin which allows you to add literal messages insted of "Out Of Stock" and "In Stock" messages.

== Description ==

This plugin allows you to supply a literal message for out of stock products and also in stock products. You can add html or link inside the message. You will never lose your potential clients again.

= Features =

* Allows text / html inside the message.
* Allows custom message for out of stock and in stock products.

== Installation ==

1. Install and activate WooCommerce.
2. Upload this plugin to the /wp-content/plugins/ directory.
3. Activate the plugin through the Plugins menu in WordPress.
4. Go to WooCommerce > Stock Message. Then type your note on "Out of Stock Message" and "In Stock Message" fields. Then Save your setting.
